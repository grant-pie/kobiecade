# KobieCade

A lightweight browser arcade built with plain **HTML/CSS/JavaScript** (no build step) featuring four mini‑games in a retro cabinet UI.

## 🎮 Included Games
- **Bubble Slumber** – pop dream bubbles to wake Kobie
- **Beast Feeder** – fling biscuits to hungry pets
- **Coffee Rush** – catch coffee and dodge hazards
- **Colour Capture** – tap the right colours before time runs out

## 🚀 Run locally
1. Open `index.html` in a web browser
2. Click a game card to play

## 🔧 Tech & Features
- Uses `<canvas>` for animated previews and simple game rendering
- High score tracking shared across games
- Works offline (no external dependencies)
- Retro arcade layout with responsive design

## 🧩 Project Structure
- `index.html` — main arcade landing page
- `landing.js` — animated game preview rendering
- `highscores.js` — shared score persistence
- `/[game]/` — each game has its own `index.html`, assets, and logic
